# E-Kart React Shopping App

A simple eCommerce cart application built using React, Vite, TypeScript, and Bootstrap.

## Features

- Product listing from DummyJSON API
- Add to cart
- Increment / decrement item quantity
- Max quantity limit (5 items per product)
- Cart thumbnail preview
- Dynamic total price calculation
- Responsive UI using Bootstrap
- Persistent navbar

## Tech Stack

React  
Vite  
TypeScript  
Axios  
Bootstrap  
Context API + useReducer

## API
https://dummyjson.com/carts
